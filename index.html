Game Instructions

Welcome to the Haunted House Game! This game is a part of a study that compares artificial and natural intelligence. We aim to measure model-based reasoning in different LLMs.

Your objective is to find the key and escape the haunted house while avoiding the ghost. Here's how to play:

  • The house is a 3x3 grid. You start some room.
  • Type the direction of the room you want to move to (up, down, left, or right)
  • If you enter a room with a ghost, it's game over!
  • If you find the key, you can escape, but first go back to the start with the key.

Type a direction to move.
