Game Instructions

Welcome to the Haunted House Game! Your objective is to find the key and escape the haunted house while avoiding the ghost. Here's how to play:

  • You start at room C1.
  • Type the code of the room you want to move to (e.g., C2, B1)
  • If you enter a room with a ghost, it's game over!
  • If you find the key, you can escape, but first go back to C1 with the key.

Type a room code to move.
